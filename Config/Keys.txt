This is a listing of all the keys that are detectable by the Forge engine.
All of these key definitions are caps sensitive.
In order to denote a keybinding requires a modifier key, format the binding such as 'Modifier+Key'.
To specify a bind's combination to be control S, the binding would be 'LeftControl+S'
Any key can be used as a modifier.
